# Conteúdo

### Modelagem de Dados:
- Entidade, Atributo e Relacionamento;
- Cardinalidade;
- Modelo Conceitual;
- Modelo Lógico;
  
### DDL (Data Definition Language):
- CREATE;
  - DB, TB e VIEW;
- ALTER;
  - MODIFY;
  - CHANGE;
  - RENAME;
- DROP;

### DML (Data Manipulation Language):
- INSERT;
- DELETE;
  - com WHERE;
- UPDATE;
  - com WHERE;

### DQL (Data Query Language):
- SELECT;
  - GROUP BY;
  - JOIN;
  - UNION.
